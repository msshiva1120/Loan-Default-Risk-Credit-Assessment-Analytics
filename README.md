# Loan-Default-Risk-Credit-Assessment-Analytics
Loan Default Risk Analysis project using Python, MySQL, and Power BI to analyze loan repayment behavior.
Performed data cleaning, validation, and exploratory data analysis on 255K+ loan records.
Developed a Risk Scoring Model to classify borrowers into different risk levels.
Built interactive Power BI dashboards to analyze credit risk, loan portfolio trends, and customer demographics.
The project helps identify key factors influencing loan defaults and supports data-driven lending decisions.

## Project Objective
1.Analyze loan portfolio and borrower characteristics.
2.Identify factors affecting loan default.
3.Perform exploratory data analysis (EDA).
4.Detect outliers and patterns in financial data.
5.Develop a risk scoring model to classify borrowers.
6.Provide insights for better credit assessment.

## Process
Collected the loan dataset and imported the data into MySQL for structured storage.
Performed data cleaning and preprocessing using Python (Pandas) by handling missing values, duplicates, and invalid data.
Conducted exploratory data analysis (EDA) using visualizations like histograms, box plots, and correlation matrices.
Developed a Risk Scoring Model to classify borrowers into different risk categories.
Built interactive Power BI dashboards to analyze loan portfolio performance and credit risk insights.

## Conclusion
  The analysis provides insights into loan performance, risk segmentation, and borrower characteristics    using multiple visualizations.Most loans are successfully repaid, while a smaller portion results in loan defaults, indicating manageable risk for the bank.Factors such as credit score, income level, DTI ratio, employment type, and age group influence the likelihood of loan default.These insights help banks improve credit risk assessment and make better lending decisions in the future.

## Insights
1.Most loans are successfully repaid, with only about 11.61% resulting in default, indicating that the
bank’s lending portfolio is generally stable.
2.High-risk borrowers contribute to more loan defaults, showing that credit risk segmentation is
important for identifying risky customers.
3.Higher Debt-to-Income (DTI) ratios are associated with increased default risk, meaning borrowers with higher financial obligations are more likely to default.
4.Employment stability influences repayment behavior, as borrowers with stable jobs (full-time) tend to have lower default rates compared to unemployed or part-time borrowers.
5.Demographic and financial factors such as age, income, credit score, and interest rate significantly
impact loan default patterns, helping banks make better credit approval decisions
